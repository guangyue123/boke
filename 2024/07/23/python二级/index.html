试题类型分析

  1. 单选题（40分）

    • 计算机常识

      • 软件的三要素是程序、数据、相关文档
      • 软件工程三要素：方法、工具、过程
      • 软件测试过程4步骤：单元测试、即成测试、验收测试（确认测试）和系统测试
      • 数据流图作用：支持软件系统功能建模
      • 结构化程序设计原则：自顶向下、逐步求精、模块化、限制使用goto语句
      • 软件需求规格说明书内容：软件的目标、软件的功能需求、性能需求、外部接口、属性及约束条件
      • 软件设计中，常用的过程设计工具有：
        1. 图形工具：程序流程图、N-S图、PAD图、HIPO图
        2. 表格工具：判定表
        3. 语言工具：PDL（伪码）
      • 每个程序都具备的统一运算模式是IPO模式
      • 解释是将源代码逐条转换成目标代码同时逐条运行目标代码的过程。执行解释的计算机程序为解释器
      • 程序语言类别
        • 机器语言（二进制代码）
        • 汇编语言（面向机器的程序设计语言）
        • 高级语言（面向过程语言、面向对象语言）

    • 数据模型

      • 层次模型：树状结构
      • 网状模型：网状结构
      • 关系模型：二维表结构
      • 概念模型：简单符号描述信息，eg：E-R图
      • 物理模型：
      • 逻辑模型：

    • E-R图

    • 二叉树

      • 对任何一棵二叉树，度为0的结点（即叶子结点）总是比度为2的结点多一个
      • 总结点数=所有节点的度之和再加1
      • 二叉树遍历
        • 前序遍历：根结点 —> 左子树 —> 右子树
        • 中序遍历：左子树—> 根结点 —> 右子树
        • 后序遍历：左子树 —> 右子树 —> 根结点

    • 算法

      • 快速排序算法的最坏情况比较次数是n(n-1)/2
      • 冒泡排序算法的最坏情况比较次数是n(n-1)/2

    • 分支结构

      • 二分支结构：if-else
      • 多分支结构：if-elif-else

    • 列表和元组可以表示一维数据，字典表示高维数据

    • Unicode编码

      • 数字0 ——> 48
      • 大写字母A ——> 65
      • 小写字母a ——> 97
      • 汉字 ——> \u4e00-\u9fa5

    • python语言的三个重要特点：

      • （1）通用性
      • （2）语法简洁
      • （3）生态高产（内置库和第三方库丰富）
      • 具体特点：
        • （1）平台无关
        • （2）强制可读
        • （3）支持中文

    • random库采用更多随机数生成算法是蒙特卡洛方法

    • 常见第三方库：

      • 网络爬虫：requests、scrapy、pyspider

      • 数据分析：numpy、pandas、scipy、seaborn

      • 文本处理：pdfminer、python-docx、beautifulsoup4

      • 数据可视化：matplotlib、seaborn、mayavi

      • 用户图形界面：PyQt5、wxPython、PyGObject、PIL、Arcade、pygtk

      • 机器学习：scikit-learn、Tensorflow、MXNet、keras、PyTorch

      • Web开发：Django、pyramid、flask

      • 游戏开发：pygame、Panda3D、cocos2d

  2. 基础编程题（40分）

    • format格式化输出

    • python库（turtle库、jieba库、random库、time库）

      • turtle库

        • 画笔对应的光标默认是在画布的==中心==，方向默认是==水平向右==

        • turtle.mainloop() # 让画布一直存在，这句代码需要放在最后

        • trule.done() # 和turtle.mainloop()作用一样

        • turtle.setup(宽度, 高度) # 设置画布的宽度和高度

        • turtle.title(标题) # 设置标题

          • turtle.pencolor(颜色) # 设置画笔画出的线的颜色
          • turtle.width(线宽) # 设置画笔线宽
          • turtle.speed(速度值) # 设置笔移动的速度(速度值是1-10逐渐变快)
          • turtle.hideturtle() #隐藏乌龟
            • ==turtle.fd(距离) # 控制笔前进指定距离==
            • ==turtle.bk(距离) # 控制笔后退指定距离==
            • ==turtle.goto(x坐标, y坐标) # 控制笔移动到指定==
          • turtle.home() # 笔回到初始状态（回到初始位置和初始方向）
          • ==turtle.left(角度) # 向左旋转指定角度==
          • ==turtle.right(角度) # 向右旋转指定角度==
          • turtle.seth(角度) # 设置绝对角度值指定度数（x轴逆时针角度）
            • ==turtle.up() # 抬起笔==
            • ==turtle.down() # 放下笔==
              • ==turtle.circle(半径) # 画一个完整的圆环==
              • ==turtle.circle(半径, 角度) # 画指定角度对应的圆弧==
              • ==turtle.dot(直径, 颜色) # 直径为指定值的画实心圆==
          • turtle.fillcolor(颜色) # 设置填充颜色
          • turtle.begin_fill() # 开始填充
          • turtle.end_fill() # 结束填充

        • turtle.write(文字内容, font=(字体名称, 字体大小, 文字类型))

          • 字体类型：normal、bold、italic

      • jieba库

        • jieba.lcut(seg_str) # 精简模式，返回一个列表类型的结果
        • jieba.lcut(seg_str, cut_all=True) # 全模式，使用 ‘cut_all=True’ 指定
        • jieba.lcut_for_search(seg_str) # 搜索引擎模式

      • random库

        • random.seed(a=None) # 随机种子
        • ==random.random() # 产生[0.0, 1.0)之间的随机浮点数==
        • ==random.randint(a, b) # 产生[a, b]之间的随机整数（a和b都可以取到）==
        • ==random.randrange(a) # 产生[0, a)之间的随机整数（a取不到）==
        • ==random.uniform(a, b) # 产生[a, b]之间的随机浮点数==
        • random.choice(seq) # 在序列类型中随机取一个元素
        • random.choices(seq,weights=None, k) # 对序列类型进行k次重复采样，可设置权重，权重小的说明被取到的概率更小
        • random.shuffle(seq) # 将序列类型中元素随机排列，返回打乱后的序列
        • random.sample(pop, k) # 从pop类型中随机选取k个元素，以列表类型返回，如果k大于所有元素的个数，则报错
        • random.gauss(mean, std) # 生产一个符合高斯分布的随机数，mean–均值，std–标准差

      • time库

        • 时间获取类：time()、ctime()、gmtime()

          • time.time() # 1970年1月1日0：00至现在的时间，单位秒，返回值：浮点数

            • import time
print(time.time())
>>> 1710820108.9800813

          • time.gmtime() # 表示为计算机可处理的时间格式，返回值：计算机内部时间变量类型

            • import time
print(time.gmtime())
>>> time.struct_time(tm_year=2024, tm_mon=3, tm_mday=19, tm_hour=3, tm_min=50, tm_sec=6, tm_wday=1, tm_yday=79, tm_isdst=0)

        • time.strptime() eg:

          • import time
timestr = "2021-01-30 11:12:46"
print(time.strptime(timestr, "%Y-%m-%d %H:%M:%S"))
>>> time.struct_time(tm_year=2021, tm_mon=1, tm_mday=30, tm_hour=11, tm_min=12, tm_sec=46, tm_wday=5, tm_yday=30, tm_isdst=-1)

        • insert(index,obj) # 将元素obj添加到列表的index位置处。

        • list1 = [1,2,3,4]
list1.insert(1,6)
print(list1)
>>> [1, 6, 2, 3, 4]

        • 列表元素删除

          • list.pop(index = -1) # 删除列表中index处的元素（默认index = -1），并且返回该元素的值
          • list.remove(obj) # 删除列表中第一次出现的obj元素
          • list.clear() # 删除列表中所有元素

        • 查找元素位置

          • list.index(x) # 函数从列表中查找某个元素(x)第一次出现的位置。

        • 列表排序

          • list.sort(reverse=True|False, key=myFunc) # 对原列表进行排序，如果指定参数，则使用比较函数指定的比较函数

        • 颠倒列表顺序

          • list.reverse() # 用于反向列表中元素

        • 删除列表

          • del(list)

        • 返回列表元素最大值和最小值

          • max(list), min(list)

        • 复制列表

          • list.copy()

      • 字典

        • 字典创建
          • {}、dict()
        • 字典中键不能重复，是唯一的，但是值可以重复
        • 字典的键只能是字符串或者数字
        • 字典的增加
          • dict[key] = value
        • 字典的删除
          • del dict[key] # 查找到字典的键删除值
          • dict.clear() # 清空字典
          • dict.pop(key) # 删除指定键所对应的键值对，返回值
          • dict.popitem() # 删除随机一个键值对，一般是最后一个键值对
        • 字典的修改
          • dict[key] = value # 查询的键不存在时报错
          • dict.update(key=value | {key: value})
        • 字典的查询
          • dict[key] # 查询的键不存在时报错
          • dict.get(key, None) # 键不存在不报错，键不存在时默认返回None，可设置返回值
          • dict.keys() # 获取所有键，返回dict_keys数据类型
          • dict.values() # 获取所有值，返回dict_values数据类型
          • dict.items() # 获取所有键值对组成的元组

      • 元组

        • 概念
          • 元组和列表相似，但是元组中的元素不允许被修改，因此元组也被称为只读列表
          • 元组不支持修改，但支持索引、拼接、成员检查、重复等相关操作
          • 元组特性之元素不允许被修改，但其元素的元素为可变类型时则支持修改。
          • 元组特性之任何无符号的对象，以逗号分割，默认被视为元组。
        • 元组的创建
          • ()、tuple()
        • 元组的其他操作和列表一样，因为元组不能修改，所以没有添加元素，删除元素等操作

      • 集合

        • 概念
          • 集合（set）是一个无序的不重复元素序列
        • 集合创建
          • 创建空集合
            • set()，不可用{}
        • 添加元素
          • set.add(x)
          • set.update(x)，x可以为列表元组字典等
        • 删除元素
          • set.remove(x)，x不存在报错
          • set.pop()，随机删除一个元素
          • set.discard(x)，x不存在不报错
        • 清空集合
          • set.clear()
